Approbations

Les demandes arrivent dans votre courriel. Collez le courriel reçu ci-dessous — tout le reste se remplit automatiquement.

Étape 1 — Traiter une demande reçue

① Vérifier le n° OPQ au registre de l'Ordre →

③ Re-téléversez ce fiche.js sur Netlify (avec les autres fichiers), puis envoyez le courriel d'approbation ci-dessous (déjà prérempli).

Étape 2a — Approuver un pharmacien

  1. Vérifiez son n° OPQ au registre de l'Ordre (bouton à l'étape 1)
  2. Téléchargez le fiche.js mis à jour (étape 1) et re-téléversez le site sur Netlify
  3. Envoyez-lui son courriel d'approbation ci-dessous (prérempli par l'analyse)
Ouvrir dans Gmail

Étape 2b — Approuver une pharmacie

  1. Validez le sérieux de la demande (bannière, adresse, téléphone)
  2. Le lien personnalisé est rempli automatiquement par l'analyse (étape 1) s'il était dans le courriel
  3. Envoyez le courriel d'approbation
Ouvrir dans Gmail

Règles du réseau (appliquées à tous les contrats)

Pour modifier ces règles : éditez le bloc REGLES dans fiche.js, puis re-téléversez le site. La pharmacie et le pharmacien peuvent offrir un taux PLUS ÉLEVÉ que le minimum, jamais moins. Per diem et hébergement s'appliquent automatiquement au-delà du seuil — personne ne peut les modifier.
